Contributions

  • Framework for real-time, extreme-scale, exact (up to discretization error) Bayesian inference and prediction
  • Area A:
    • Spectral analysis of \(\mathbf{K}\) for elliptic-operator and data-driven priors
    • Analysis of inverse problem learning Matérn hyperparameters from data
    • Analysis of generative AI methods to augment training data distribution
    • Development and analysis of optimal experimental design problem
  • Area B:
    • Fast block-triangular Toeplitz matvec algorithm and implementation
    • Appropriate algorithm for inverting \(\mathbf{K}\)
    • Cost and communication model for matvec algorithms
    • Software release
  • Area C:
    • Application to tsunami early warning for CSZ
    • Using real bathymetry data to construct mesh
    • Applications to other problems (e.g. contaminant transport)

Closing Remarks

  • Summary
    • Exploit autonomous structure to enable real-time inference at extreme scale
    • Fast matvec algorithm speeds up Hessian action by ~44,000x
    • Exact inference and prediction with ~132M parameters and ~25K data points in 0.06s
