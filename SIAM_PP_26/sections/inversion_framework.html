Forward Problem

  • Mixed formulation for velocity and pressure unknowns1: \[\footnotesize \rho \frac{ \partial\vec{u}}{\partial t} + \nabla p = \vec{0} \quad \text{and} \quad \frac{1}{K} \frac{ \partial p}{\partial t} + \nabla \cdot \vec{u} = 0 \quad \text{in }\Omega \times (0,T) \]
  • Boundary conditions:
    • ODE type BC (sea surface): \[\small p = \rho g \eta, \frac{ \partial\eta}{\partial t} = \vec{u} \cdot \vec{n} \text{ on } \Gamma_\text{surface} \times (0,T) \]
    • Seafloor velocity (boundary source): \[\small \vec{u} \cdot \vec{n} = -\frac{\partial b}{\partial t} \text{ on } \Gamma_{\text{bottom}} \times (0,T) \]
    • First order absorbing BC (outgoing waves): \[\small \vec{u} \cdot \vec{n} = -\frac{1}{\rho c} p \text{ on }\Gamma_\text{lateral} \times (0,T), c = \sqrt{K/\rho} \]
  • Homogeneous initial conditions

Implemented in MFEM with high-order finite elements and RK4 time-stepping. 1Lotto and Dunham (2015)

Inverse Problem

  • Given observations \(d\) and forward model \(\mathcal{F}\),
    infer model parameters \(m\) — here:
    • Model parameters \(m\): spatiotemporal seafloor motion \(\frac{\partial b}{\partial t} (x,t)\)
    • Observations \(d\): pressure sensor readings \(\left\{p(x_i, t)\right\}_{i=1}^{N_d}\)
    • Parameter-to-Observable (p2o) Map: \(\mathcal{F}: m(x,t) \mapsto d(x_i,t)\)
  • Inverse problem is ill-posed
    • Observations are usually sparse and noisy
    • Many different parameters may be consistent with the data
  • Bayesian approach to ill-posedness
    • Characterize the probability that any parameter field is consistent with the data and prior knowledge
    • The uncertain parameter is treated as a random field, and the solution of the inverse problem becomes a probability density, the posterior distribution
    • Bayes' theorem: \( \pi_{\text{post}} (m|d) \propto \pi_{\text{prior}} (m) \pi_{\text{like}} (d|m) \)

Bayesian Framework, Linear Inverse Problem

Gaussian Prior with Matérn Covariance:\[m \sim \mathcal{N}(m_{\text{prior}}, \Gamma_{\!\text{prior}}), \quad \Gamma_{\!\text{prior}} \coloneqq \left( \alpha_1 I - \alpha_2 \Delta_{\text{2D}} \right)^{-2}\]

Likelihood: \(\pi_{\text{like}}(d|m) \propto \exp\left(-\frac{1}{2} \|\mathcal{F}m - d\|_{\Gamma_{\!\text{noise}}^{-1}}^2\right)\)

Observations: \(d = \mathcal{F}m + \nu, \quad \nu \sim \mathcal{N}(0, \Gamma_{\!\text{noise}})\)

Posterior

\[\begin{align*} \mu_{\text{post}} &= \mathcal{N}(m_{\text{map}}, \Gamma_{\!\text{post}}) \\ \Gamma_{\!\text{post}}&=\left(\mathcal{F}^* \Gamma_{\!\text{noise}}^{-1} \mathcal{F} + \Gamma_{\!\text{prior}}^{-1}\right)^{-1}\\ m_{\text{map}}&=\Gamma_{\!\text{post}}\left(\mathcal{F}^* \Gamma_{\!\text{noise}}^{-1} d + \Gamma_{\!\text{prior}}^{-1} m_{\text{prior}}\right) \end{align*}\]

Predictive

\[\begin{align*} \mu_{\text{post}(q)} &= \mathcal{N}(q_{\text{map}}, \Gamma_{\!\text{post}(q)}) \\ \Gamma_{\!\text{post}(q)} &= \mathcal{F}_q \Gamma_{\!\text{post}} \mathcal{F}_q^*\\ q_{\text{map}} &= \mathcal{F}_q m_{\text{map}} \coloneqq Q d \end{align*}\]

Posterior covariance does not depend on data. subset of the Matern family of covariance operators can be written as the inverse of an elliptic different operator, with the coefficients \(\alpha_i\) chosen to impose the desired correlation and variance structure
\(\gamma \gt 3/2\) for bounded pointwise variance since parameter is a 2+1D field

Real-Time Inference and Prediction Framework

  • Offline–Online Decomposition phases:
    • Phase 1 (Offline): Construct p2o map from adjoint PDE solves
    • Phase 2 (Offline): Compute compact representation of \(\Gamma_{\!\text{post}}\)
    • Phase 3 (Offline): Compute QoI Uncertainty and Data-to-QoI map
    • Phase 4 (Online): Compute MAP point and QoI in real time
  • Enables real-time, high-fidelity inference/prediction for problems governed by linear autonomous dynamical systems1
