Goal-Oriented Dimension Reduction: Motivation

  • The preceding inversion framework relies on the linear acoustic-gravity equations
    • Can't propagate tsunami all the way to shore (inundation)
  • Idea: use inferred seafloor motion from acoustic-gravity-based framework as input to nonlinear shallow-water equations (SWE)
    • Solving the SWE in real-time is computationally difficult at large scale
    • Instead, train a surrogate (e.g., neural network) to learn mapping from seafloor motion to coastal QoIs
  • Parameterizing a neural network with 1-billion parameters is computationally infeasible
    • Traditional low-rank decompositions using SVDs of $\mathbf{F}$ have slow spectral decay
    • Autoencoders also face the dimensionality issue

Goal-Oriented Dimension Reduction: Motivation

Eigenvalues of the "inside-out" prior-preconditioned Hessian of the data misfit (\(\mathbf{F}\Gamma_{\text{prior}}\mathbf{F}^*\)) do not decay fast enough for traditional dimension reduction techniques.

Goal-Oriented Dimension Reduction: Motivation

  • Parameterizing a neural network with 1-billion parameters is computationally infeasible
    • Traditional low-rank decompositions using SVDs of $\mathbf{F}$ have slow spectral decay
    • Autoencoders also face the dimensionality issue
  • Idea: create a goal-oriented reduced basis for the parameter field
    • Captures parameter modes that are informative for the QoI
    • Used to parameterize surrogate for SWE
  • Relevant prior work: Lieberman and Willcox (2012). Goal-Oriented Inference: Approach, Linear Theory, and Application to Advection Diffusion. SIAM Journal on Scientific Computing.
    • Similar motivation and purpose, but different methodology
    • This work uses the infinite-dimensional setting to derive the reduced basis
    • The reduced space inverse problem is solved in the QoI space instead of the data space

Problem Formulation

  • Banach Spaces: $\mathcal{X}$ (parameter $m$), $\mathcal{Y}$ (data $d$), $\mathcal{Z}$ (QoI $q$).
  • Bounded Linear Maps:
    • Parameter-to-Observable (p2o): $F: \mathcal{X} \to \mathcal{Y}$
    • Parameter-to-QoI (p2q): $F_q: \mathcal{X} \to \mathcal{Z}$
  • Assuming Gaussian prior $\mathcal{N}(m_0,\Gamma_{\!\text{prior}})$ and likelihood, the posterior is $\mathcal{N}(m_{\text{map}}, \Gamma_{\!\text{post}})$:
\[\footnotesize \begin{align*} \Gamma_{\!\text{post}} &= \left(F^*\Gamma_{\!\text{noise}}^{-1} F + \Gamma_{\!\text{prior}}^{-1}\right)^{-1}\\ m_{\text{map}} &= \Gamma_{\!\text{post}} F^*\Gamma_{\!\text{noise}}^{-1} d \end{align*} \]
  • The posterior predictive measure on the QoI is the pushforward through $F_q$, yielding $\mathcal{N}(q_{\text{map}}, \Gamma_{\!\text{post}(q)})$:
\[\footnotesize \begin{align*} q_{\text{map}} &= F_qm_{\text{map}} \eqqcolon Qd \\ \Gamma_{\!\text{post}(q)} &= F_q \Gamma_{\!\text{post}} F_q^* \end{align*} \]

Spectral Analysis of the QoI Space

  • Let $\mathcal{H}_q$ be the Cameron-Martin space associated with the posterior predictive measure.
  • The covariance $\Gamma_{\!\text{post}(q)}$ maps $\mathcal{Z}^* \to \mathcal{Z}$ by: \[\footnotesize \langle \Gamma_{\!\text{post}(q)} \ell_1 ,\ell_2\rangle = \int_{\mathcal{Z}} \langle z-q_{\text{map}}, \ell_1\rangle \langle z-q_{\text{map}}, \ell_2\rangle d\mu_{\text{post}(q)} \]
  • Restricted to $\mathcal{H}_q$, $\Gamma_{\!\text{post}(q)}$ is a compact, positive, self-adjoint, nuclear (trace-class) operator.
  • Thus, we have an orthonormal eigenbasis $\{v_k\}_{k=1}^\infty$ for $\mathcal{H}_q$:
\[\footnotesize \Gamma_{\!\text{post}(q)} v_k = \lambda_k v_k, \quad \text{with} \quad \sum_k \lambda_k < \infty \]

QoI-Informative Subspace

  • Define modes $u_k \in \mathcal{X}$ as $u_k \coloneqq \Gamma_{\!\text{post}} F_q^* v_k$.
  • These are generalized eigenvectors of the pair $(F_q^*F_q,\Gamma_{\!\text{post}}^{-1})$: \[\footnotesize F_q^*F_q u_k = F_q^*F_q \Gamma_{\!\text{post}} F_q^* v_k = F_q^*\Gamma_{\!\text{post}(q)} v_k = \lambda_k F_q^* v_k = \lambda_k \Gamma_{\!\text{post}}^{-1} u_k \]
  • The modes $\{u_k\}_{k=1}^\infty$ are orthonormal in $\mathcal{H}_{\text{post}}$: \[\footnotesize \langle u_i, u_j \rangle_{\mathcal{H}_{\text{post}}} = \langle v_i, \Gamma_{\!\text{post}(q)} v_j \rangle = \lambda_j \delta_{ij} \]
Goal-Oriented Subspace

\[ \text{span}\{u_k\} = \mathcal{N}(F_q)^\perp \cap \mathcal{H}_{\text{post}} \]

The $\{u_k\}$ form a basis for the exact subspace of $\mathcal{H}_{\text{post}}$ that is informative for the QoI.

Reduced Space Inverse Problem

  • Now that we have a basis for the QoI-informative subspace, how do we use it to solve the inverse problem?
  • Projection operators: $U_K \xi = \sum_{k=1}^K \xi_k u_k$ and $V_K \xi = \sum_{k=1}^K \xi_k v_k$.
  • Now, we can project \(F, F_q, \Gamma_{\text{\!prior}}^{-1}\) to the reduced space: \[\small \begin{align*} F_{\text{proj}} &= F U_K \\ F_{q,\text{proj}} &= F_q U_K \\ \Gamma_{\text{\!prior},\text{proj}}^{-1} &= U_K^* \Gamma_{\text{\!prior}}^{-1} U_K \end{align*} \]
  • Then, the reduced-space Hessian is given by: \[ \small \begin{align*} H_{\text{red}} &= (FU_K)^*\Gamma_{\text{\!noise}}^{-1}(FU_K) + U_K^*\Gamma_{\text{\!prior}}^{-1}U_K = U_K^* \Gamma_{\!\text{post}}^{-1} U_K = H_{\text{proj}}\\ m_{\text{map, red}} &= H_{\text{red}}^{-1} (FU_K)^*\Gamma_{\text{\!noise}}^{-1} d \end{align*} \]

Reduced Space Inverse Problem

  • But we know $U_K = \Gamma_{\!\text{post}} F_q^* V_K$: \[\small \begin{align*} \textcolor{bf5700}{H_{\text{red}}} &= V_K^* F_q \Gamma_{\!\text{post}} \left[ \Gamma_{\!\text{post}}^{-1} \right] \Gamma_{\!\text{post}} F_q^* V_K = V_K^* \Gamma_{\!\text{post}(q)} V_K = \textcolor{bf5700}{D_K} \\ \textcolor{bf5700}{m_{\text{map, red}}} &= H_{\text{red}}^{-1} (FU_K)^*\Gamma_{\!\text{noise}}^{-1} d = D_K^{-1} (V_K^* F_q \Gamma_{\!\text{post}} F^*) \Gamma_{\!\text{noise}}^{-1} d = \textcolor{bf5700}{D_K^{-1} V_K^* \underbrace{Q d}_{q_{\text{map}}}} \end{align*} \]
  • This is equivalent to the following optimization problem: \[ \small \begin{align*} m_{\text{map, red}} &= \arg\min_{\tilde{m} \in \mathbb{R}^K} \frac{1}{2}\|F_{q, \text{proj}} \tilde{m} - q_{\text{map}}\|_{\Gamma_{\text{\!post}(q)}^{-1}}^2 \end{align*} \] where the "data" is the QoI \(q_{\text{map}}\), with "noise covariance" $\Gamma_{\text{\!post}(q)}$
Reduced Space Solution
\[\footnotesize \begin{gather*} q_{\text{red}} = V_K V_K^* q_{\text{map}}, m_{\text{map, red}} = D_K^{-1} V_K^* q_{\text{map}}, \\ \Gamma_{q, \text{post, red}} = V_K D_K V_K^*\ \end{gather*} \]

Error Analysis

  • The full-space vs. reduced-space QoI error is bounded by the truncated spectral tail
  • Mean error: $\|q_{\text{map}} - q_{\text{red}}\| \leq \sum_{k=K+1}^\infty |\langle q_{\text{map}}, v_k\rangle_{\mathcal{H}_q}|^2$
  • Covariance error: $\text{Tr}(\Gamma_{\!\text{post}(q)} - \Gamma_{q,\, \text{post, red}}) = \sum_{k=K+1}^\infty \lambda_k$
  • Total Error: 2-Wasserstein Distance \[\small W_2^2 \leq \sum_{k=K+1}^\infty \lambda_k + |\langle q_{\text{map}}, v_k\rangle_{\mathcal{H}_q}|^2 \]

Note: This analysis holds generally and does not assume $F$ or $F_q$ arise from autonomous dynamical systems.

Computational Framework

  • Options: solve generalized eigenvalue problem for \((\mathbf{F} _q^*\mathbf{F}_q, \mathbf{\Gamma}_{\text{\!post}}^{-1})\)
    or standard eigenvalue problem for \(\mathbf{\Gamma}_{\text{\!post}(q)}\)
    • First option may be suitable for some problems using, e.g., LOBPCG
    • For problems governed by autonomous dynamical systems, the second option is more efficient
  • Computing the dense operators $\mathbf{Q}$ and $\mathbf{\Gamma}_{\text{\!post}(q)}$ requires solving the full-space inverse problem many times
    \(\to\) fast inversion framework and FFT-based matvecs for $\mathbf{F}, \mathbf{F}_q$
  • For $\mathcal{O}(10^2)$ sensors and $\mathcal{O}(10^1)$ QoI locations, the dense operators fit onto a single GPU
    • The reduced-space inverse problem can be solved directly via the dense discrete operators
